# 📝 To-Do List App

A beautiful and intuitive To-Do List mobile application built with Expo React Native and TypeScript.

## 🛠 Tech Stack

- **Frontend**

  - Expo React Native
  - TypeScript
  - Redux Toolkit
  - React Navigation

- **Backend**
  - Node.js
  - Express
  - MongoDB
  - JWT Authentication

## 🚀 Getting Started

1. **Clone the repository**

```bash
git clone [your-repo-url]
cd to-do-list-app
```

2. **Install dependencies**

```bash
cd app
npm install
```

3. **Start the development server**

```bash
npm start
```
